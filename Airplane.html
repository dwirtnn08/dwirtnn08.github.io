Deskripsi Variabel :

  • satisfaction : tingkat kepuasan penumpang pesawat (Satisfaction, Dissatisfaction)
  • Gender : jenis kelamin customer (Female, Male)
  • Customer.Type : jenis pelanggan (Loyal customer, Disloyal customer)
  • Age : usia penumpang
  • Type.of.Travel : Tujuan penerbangan penumpang (Personal Travel, Business Travel)
  • Class : Kelas perjalanan di pesawat penumpang (Bisnis, Eco, Eco Plus)
  • Flight.Distance : Jarak penerbangan dari perjalanan ini
  • Seat.comfort : Tingkat kepuasan kenyamanan kursi
  • Departure.Arrival.time.convenient : Tingkat kepuasan waktu Keberangkatan/Kedatangan yang nyaman
  • Food.and.drink : Tingkat kepuasan Makanan dan minuman
  • Gate.location : Tingkat kepuasan lokasi Gerbang
  • Inflight.wifi.service : Tingkat kepuasan layanan wifi dalam pesawat (0: Tidak Berlaku; 1-5)

# prediktor data train
train_x <- air_train %>% select_if(is.numeric)

# target data train

# prediktor data test
test_x <- air_test %>% select(-c(satisfaction, Customer.Type, Class, Type.of.Travel, Gender, predAir, predAir.Label))

# target data test

Melakukan scaling untuk data train dan test

train_x <- scale(train_x)

                center = attr(train_x, "scaled:center"), #nilai rata-rata train

Conclusion

Dapat dilihat pada 2 model diatas modelAir_step dan airPred_KNN menggunakan algoritma yang berbeda.Pada model modelAir_step menggunakan Logistic Regresion sebagai algoritma dalma pembuatan model.Sedangkan model airPred_KNN menggunakan algoritma KNN (K-Nearest Neighbor) menghasilkan hasil yang cukup baik dalam memprediksi apakah customer puas atau tidak dengan pelayanan dari maskapai penerbangan.Dapat disimpulkan bahwa model KNN dalam memprediksi kelas target lebih baik dari pada model Logistic Regression karena memiliki nilai precission sebesar 91%.
